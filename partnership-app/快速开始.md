# 🚀 快速开始指南

## 一键启动（完整流程）

### 步骤 1：安装依赖（第一次运行）

**前端依赖（工作区已运行）：**
```bash
# 前端已运行在 http://localhost:5175
# 如果需要重启：
npm run dev
```

**后端依赖：**
```bash
cd backend
npm install
```

### 步骤 2：启动后端

**在 backend 目录：**
```bash
node server.js
# 或使用脚本：start-backend.bat
```

看到输出：
```
✅ Database initialized
🚀 Server running on port 3000
📱 Frontend: http://localhost:5175
🔌 WebSocket ready
```

### 步骤 3：多人协作测试

1. 你自己打开：`http://localhost:5175`
2. 用浏览器无痕模式或另一台电脑打开：`http://localhost:5175`（或局域网 IP）
3. 在一侧添加/修改待办事项/项目/合作方
4. **另一侧会实时看到更新！** ✨

---

## ❓ 故障排查

| 问题 | 解决方案 |
|------|----------|
| WebSocket 连接失败 | 确保后端 server.js 已启动在 3000 端口 |
| 前端显示 404/500 | 检查后端是否运行，刷新页面 |
| 无实时更新 | 打开浏览器开发者工具(Console)，查看 WebSocket 连接状态 |
| 端口冲突 | 修改 vite.config.js 的 port 或 server.js 的 PORT |

---

## 📌 重要说明

1. **数据存储**：所有数据保存在 `backend/partnership.db`（SQLite 单文件）
2. **网络环境**：
   - 同一台电脑：`localhost`
   - 局域网：用 `你的IP:5175`
3. **认证**：当前版本跳过认证（所有访问者可以修改数据）
4. **生产部署**：需要添加用户认证、权限管理、HTTPS、数据库迁移等
